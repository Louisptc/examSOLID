Library Management

## Description

A librery management  project

## Installation

1. Clone the repository:
    
    ```bash
    git clone https://github.com/Louisptc/examSOLID.git
    ```

2. Navigate to the project directory:
    
    ```bash
    cd exaSOLID
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

4. Generate the database:

    ```bash
    npm run start-db
    ```

## Build

1. Build the project:
    
    ```bash
    npm run build
    ```

2. Start the server:
    
    ```bash
    npm run start
    ```
